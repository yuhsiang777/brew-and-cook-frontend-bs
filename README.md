# Brew & Cook 前端頁面（Bootstrap 5）

本專案為 Brew & Cook 精品廚具電商的前端介面，使用 HTML + Bootstrap 5 架設，搭配自建 API 串接商品資料、會員登入與購物功能。

## 🔧 技術棧

- HTML5
- CSS3 / Bootstrap 5
- JavaScript（純 JS）
- Axios（API 串接）
- GitHub Pages（可選擇部署）

## 📁 專案結構

```bash
brew-and-cook-frontend-bs/
├─ index.html        # 首頁
├─ products.html     # 商品列表
├─ login.html        # 登入頁
├─ register.html     # 註冊頁
├─ css/styles.css    # 自訂樣式
├─ js/scripts.js     # 自訂 JS
